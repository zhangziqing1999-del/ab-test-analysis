# ab-test-analysis

多轮交互类产品（对话式AI、智能推荐）线上AB实验的一站式分析工具。输入一份实验日志，自动产出实验组 vs 对照组的转化率、完读率对比与显著性检验，并支持对用户query做大模型打标后分维度下钻。

## 特性

- **2-5组对比**：一次分析多个实验组
- **双指标体系**：多轮交互转化率 + 内容完读率，自动做比例Z检验
- **query智能打标**：大模型自动标注 主观/客观 + 18类汽车场景
- **分维度下钻**：看不同类型/场景query下的实验效果差异
- **高性能**：query去重(省~50%) + 多模型并发，提速5-6倍
- **断点续传**：任务中断重跑自动恢复，不丢数据

## 快速开始

```bash
python3 scripts/ab_label_analysis.py \
  --file "实验数据.csv" \
  --groups "1,2" \
  --labels "对照组,实验组" \
  --exp-id 225457
```

运行后在数据同目录生成 `AB分析结果_标签维度_<文件名>.xlsx`。
